Игры для тренировки устного счёта.

## Системные требования

 - Bash / Zsh
 - Make
 - Git
 - Node 13+

## Установка

Для установки выполните следующие команды:

```bash
git clone git@github.com:julish13/frontend-project-lvl1.git
cd frontend-project-lvl1
make
npm link
```
